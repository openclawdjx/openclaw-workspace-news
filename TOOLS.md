# TOOLS.md - Tools & Capabilities

## 搜索工具

### Web 搜索（必须使用）
- **minimax_search** - MiniMax MCP 搜索（获取实时新闻）
- **multi_search_engine** - 多搜索引擎（百度/Google/Bing等）

### 浏览器
- **browser** - 浏览器控制，用于复杂网页交互
- **重要**：使用完毕后必须关闭浏览器页面

## 文件操作
- **Read** - 读取文件内容
- **Glob** - 按模式搜索文件
- **Grep** - 搜索文件内容

## 终端
- **Bash** - 执行shell命令

## Obsidian
- **notesmd-cli** - 存入Obsidian笔记，路径为 vault 下的 news 文件夹
